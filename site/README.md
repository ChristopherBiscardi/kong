# Mashape Kong Frontend

This is the source of the frontend code.

## Requirements

- Bundler
- NPM
- Bower

## Install

All bash commands must be executed in `./site` directory.

>```bash
npm install
>```

## Running Locally

>```bash
gulp
>```

## Building

For production:

>```bash
gulp build:prod
>```

## Deploy

This will deploy to github pages.

>```bash
gulp deploy:prod
>```
