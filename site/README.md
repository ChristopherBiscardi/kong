# Mashape Kong Frontend

This is the source of the frontend code.

## Requirements

- Bundler
- NPM


## Install

for linux systems:

> ```bash
sudo apt-get install ruby-dev
> ```

>```bash
npm install
>```


## Running Locally

>```bash
gulp
>```


## Building

For staging:

>```bash
gulp build:staging
>```

For production:

>```bash
gulp build:prod
>```

## Deploy

This will deploy to github pages.

>```bash
gulp deploy:prod
>```
